# 网文创作助手

一个基于AI的网文创作辅助工具，集成了智能对话、文档编辑、项目管理和AI工具等功能，旨在提升网文创作效率和质量。

## 开发说明

此工具使用Python开发，通过API与各大AI服务商集成，实现智能化的网文创作辅助功能。

## 功能特性

### 1. AI智能对话系统
- 支持多种AI提供商（OpenAI、Anthropic）
- 集成工具调用功能，可读取/写入项目文件
- 技能系统，支持自定义触发关键词激活特定能力
- 记忆压缩机制，保持长对话上下文

### 2. 项目管理
- 多项目管理，支持项目创建、重命名、删除
- 项目包含世界观、人物库、章节纲要、章节内容
- 自动保存功能，防止数据丢失
- 项目导入/导出功能

### 3. 文档编辑器
- 支持查找替换
- 字数统计功能
- 错别字检查
- 历史版本管理，可查看和恢复历史版本
- 支持撤销/恢复操作
- 全局查找替换功能

### 4. AI增强工具
- **取名工具**：生成人物、地点、功法等名称
- **风格蒸馏**：分析文本风格并生成模仿指导
- **拆书工具**：自动生成章节纲要和情节总结

### 5. 技能管理系统
- 自定义技能模板
- 支持导入/导出技能
- 技能启用/禁用管理
- 触发关键词自动激活

### 6. 章节管理
- 章节创建、重命名、删除
- 章节列表展示
- 自动排序功能

Windows 7+ 64位

## 注意事项

1. **卡密验证**：首次使用需要输入有效卡密，卡密购买地址：https://hkyy.pp.ua
2. **AI API**：需要有效的AI服务API Key
3. **自动保存**：每30秒自动保存一次当前章节
4. **历史版本**：最多保留10个历史版本
5. **网络连接**：AI功能需要稳定的网络连接
